# 📊 Airdrop Income Tracker

Aplikasi web sederhana untuk mencatat, melihat, dan menganalisis income dari airdrop berbasis bulan dan tahun.

## CARA INSTALL

npm install express
npm init -y
node server.js

Akses di browser : 
http://localhost:3000
